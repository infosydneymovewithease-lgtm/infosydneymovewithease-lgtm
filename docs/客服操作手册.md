# 迁喜搬家 · 客服操作手册

> 适用对象：客服 / 调度员
> 最后更新：2026-05-07
> 后台地址：**https://qianxibanjia.com.au/login**

---

## 0. 登录

1. 打开 **https://qianxibanjia.com.au/login**
2. 输入分配给你的账号 + 密码 → 点登录
3. 系统自动跳到「工作台」页面

> 🔒 **登不上怎么办**：
> - 复制粘贴密码（别手敲，容易漏字符）
> - 浏览器**强制刷新**：Mac 按 `Cmd+Shift+R`，Windows 按 `Ctrl+Shift+R`
> - 浏览器自动填的可能是旧密码 → 手动改输新密码
> - 都不行 → 找管理员重置密码

## 1. 工作台（首页）

登录后第一个页面，最重要的两块：

| 区块 | 说明 |
|---|---|
| 🔴 **需要处理** | 显示需要客服马上处理的事项（红色警示）。每天上班第一件事就是清零这里 |
| 📋 **今日订单** | 今天有哪些单要执行（提醒派单 / 跟进） |
| 📅 **最近 7 天** | 历史订单一览 |

> 💡 把「需要处理」的项点完一遍，这一天的核心 KPI 就完成大半。

## 2. 日常工作流（按发生频率排序）

### 2.1 客户咨询 → 录订单（电话/微信打过来的单）

**菜单：左边「新建订单」**

1. 点 **新建订单**
2. 选服务类型：搬家 / IKEA 提货 / 清洁 / 寄存 / 二手物品
3. 填客户基本信息（**手机号必填，是后续查单的钥匙**）
4. 填地址（用 Google 地址搜索框，会自动算距离）
5. 选车型（系统会显示**建议报价**，可以直接采用或手动改）
6. 选时段（系统会自动显示当天可用时段）
7. **❗ 必问重物**：录单时主动问客户「有没有钢琴 / 双开门冰箱 / 大理石 / 保险箱 / 跑步机 / 商用大件 / 实木大衣柜 / 麻将机 / 玻璃易碎品」—— 这些都要单独加重物费（详见 **第 4 节 重物费内部价目表**）
8. **定金状态**：
   - ✅ **已收定金**（推荐）→ 创建后状态就是「已收定金」
   - ⏳ **待定** → 状态「待确认」，提醒后续跟进
   - ❌ **未付款** → 不能创建（系统会拦住）
9. 点「创建订单」→ 完成

> 💰 **定金金额**：澳洲账户 **$30 AUD**，国内支付宝 **¥150 RMB**。

---

### 2.2 处理客户网站自助下单（带「待确认」标签的单）

客户在 `qianxibanjia.com.au` 自己下的单，会进系统，状态是「**待确认**」。

**菜单：左边「全部订单」→ 筛选状态「待确认」**

每张「待确认」单都要做 **3 件事核实**，**全部完成才能派单**：

1. ☎️ **联系客户**（电话或微信，确认人对得上）
2. ⏰ **时间确认**（确认到达时段）
3. 📍 **地址确认**（核对取货 + 卸货地址）

**操作位置**：点订单进详情页 → 顶部能看到 ✅ 三个勾选框。
都打勾后，状态自动变成「已收定金」，可以派单。

> ⚠️ **没核实的单不能派**！系统会提示「请先完成核实（联系客户 + 时间确认 + 地址确认），才能派单」。这是为了防止 IP 单（错地址、错时间）派出去后客户拒收。

---

### 2.3 派单（把订单分给师傅）

**菜单：左边「派单管理」**

1. 看到所有「已收定金」的待派单
2. 找到要派的单 → 选可用师傅 → 点派单
3. 派完师傅那边手机会立刻看到这单（在他的师傅端）

或者直接进订单详情：**全部订单 → 点订单 → 选师傅 → 派单**。

---

### 2.4 修改订单

**菜单：全部订单 → 点订单进详情 → 编辑**

可以改的：
- 时间 / 时段
- 地址（注意改完会重新算距离）
- 车型（会触发重新报价）
- 备注
- 报价（最终金额，客服可手动调整）

> ✏️ 改完会留下修改痕迹（谁改的、什么时候改的），方便后期追踪。

---

### 2.5 取消订单

**菜单：全部订单 → 点订单 → 取消**

要填**取消原因**（客户改主意 / 客户失联 / 时间冲突 / 其他）。
取消后订单进入「**取消历史**」，不出现在主列表。

---

### 2.6 查询客户历史订单

**最快方法：左边「全部订单」→ 顶部搜索框输入手机号**

或者点订单详情，里面有「该客户历史订单」一栏。

> 📞 客户来电话查单，**永远先问手机号**，是最快的关联钥匙。

## 3. 后台功能详解（按菜单逐页讲）

> 客服可见的菜单从上到下共 11 个，下面一个一个讲。
> 「客户主档」「工资结算」「运力管理」「系统设置」是**管理员专属**，客服看不到。

---

### 3.1 工作台

**入口**：左边第一个菜单（也是登录后的默认页）

**这个页面是干什么的**：每天上班第一眼看的「驾驶舱」，告诉你「现在什么事最急」。

**页面分 4 块（从上往下）：**

| 区块 | 显示什么 | 怎么用 |
|---|---|---|
| 🔴 **需要处理** | 待派单订单数、未付款订单数等红色警示 | 点过去直接跳到对应订单列表 |
| 💰 **今日收入** | 今天已完成订单的实收金额合计 | 看一眼即可 |
| 📋 **今日订单** | 今天要执行的所有订单 | 点单进详情，跟进派单/确认 |
| 📅 **最近 7 天** | 7 天内的订单数 + 收入趋势 | 月底盘账时参考 |

**核心动作**：每天来上班，先把「**需要处理**」清零（数字归零或者「暂无待处理事项」），这一天的 KPI 就稳了。

---

### 3.2 新建订单

**入口**：左边「新建订单」

**这个页面是干什么的**：客服接到客户咨询（电话/微信/上门）后，手动录入订单。

**支持的服务类型（顶部切换）：**
- 🚚 **搬家**（最常用）
- 🛋️ **IKEA 提货**（IKEA 大件配送/安装）
- 🧹 **清洁服务**
- 📦 **寄存**（短期/长期物品寄存）
- ♻️ **二手物品**（接收二手家具）

**关键字段（按顺序填）：**

1. **客户姓名 + 手机号**（手机号必填，必须！）
2. **服务地址**（Google 自动补全，会自动计算距离 + 远途费）
3. **车型 + 时段**（系统按车型自动显示该天可用时段，不可用的会灰掉）
4. **重物 / 楼梯 / 备注**（可选，但有的话一定要填）
5. **建议报价**（系统自动算）→ 客服可以**手动覆盖**最终价格
6. **定金状态**（必选）：
   - ✅ 已收定金 → 直接创建为「已收定金」
   - ⏳ 待定 → 创建为「待确认」状态
   - ❌ 未付款 → 系统不让你提交

**注意事项：**
- 录单**必须问重物**（参考第 4 节价目表），不然师傅到现场要加费容易跟客户扯皮
- 录单时间和服务时间是两回事 —— 选「服务日期」别选错了

---

### 3.3 全部订单

**入口**：左边「全部订单」（客服**最高频**使用的页面）

**这个页面是干什么的**：所有「搬家」订单的集中列表 + 详情入口。

**顶部工具栏：**
- 🔍 **搜索框**：输入客户名 / 手机号 / 订单号即时搜索
- 📅 **按月查看**：选某个月，只看那个月的订单
- ⬆️⬇️ **排序**：录单时间 / 服务日期，最新或最早
- 🏷️ **状态筛选标签**（顶部一排横向标签）：
  - **待确认**（带数字徽章）→ 客户自助下单 + 客服录单未确认的
  - **已收定金** → 已确认但还没派单
  - **已派单** → 已分配师傅，等出发
  - **进行中** → 师傅已经开始工作（计时中）
  - **已完成** → 师傅填完账单
  - **已取消** → 已取消（也会去「取消历史」页）

**列表每行显示：**
- 订单号 + 服务日期 + 时段
- 客户姓名 + 手机号
- 车型 + 距离
- 报价 + 定金状态（「定金 $30」或「定金未收」）
- 派单情况（「未派单」或显示师傅名）

**点订单进详情页**，能做的：
- 改时段 / 改地址 / 改车型 / 改报价
- 做 3 件核实勾选（联系客户 / 时间 / 地址）
- 派单（选师傅）
- 写客服内部备注（**客户看不到**，纯客服之间留痕用）
- 取消订单
- 看该客户的历史订单

**注意事项：**
- 「待确认」单一定要先核实再派，没核实派出去会出事
- 改完订单务必看一眼右上角是否真的保存了（系统有时会卡）

---

### 3.4 取消历史

**入口**：左边「取消历史」

**这个页面是干什么的**：所有「已取消」状态的订单归档。主列表里看不到，要查就来这里。

**功能：**
- 🔍 搜索（同「全部订单」）
- 📅 按月筛选
- 看每张取消单的**取消原因**（客户/客服取消时填的）

**注意事项：**
- 这里有「**彻底删除**」按钮 —— 慎用！删了就找不回来。一般取消保留即可，不需要彻底删
- 月底统计取消率时来这里查

---

### 3.5 派单管理

**入口**：左边「派单管理」

**这个页面是干什么的**：集中处理「需要派师傅」的订单 + 看每个师傅的出勤情况。

**页面布局：**
- **左边**：待派单订单列表（按服务日期排序）
- **右边**：「师傅出勤情况」—— 每个师傅今天 / 明天 / 后天的状态
  - **空闲**（绿色）→ 可派
  - **当天已有 X 单**（黄色）→ 看是否能再加一单
  - **全部已派**（灰色）→ 当天满了

**操作：**
1. 点左边某张订单
2. 右边师傅列表里勾选要派的师傅（可勾多个，第一个是「**主带师傅**」）
3. 点确认派单

**关键概念：主带师傅 vs 副手**
- 第一位 = **主带**（开车 + 主操作 + 跟客户对接）
- 后面的 = 副手（搬运辅助）
- **2 人团队 / 3 人团队**车型必须派对应数量的师傅

**注意事项：**
- 派的是**已确认**的单（待确认的派不了，系统会拦）
- 派完师傅手机端立刻能看到（无需通知）

### 3.6 企业客户（B2B）

**入口**：左边「企业客户」

**这个页面是干什么的**：管理跟你们长期合作的公司客户（连锁店搬迁、办公室搬家等）。

**页面布局：**
- 顶部搜索：公司名 / 联系人
- 列表显示：合作中的企业 + 月均订单数
- 点公司进详情：能看该公司的所有历史订单

**操作：**
- 「**新增企业**」按钮 → 录入新合作公司基本信息
- 进公司详情可以「**新建订单**」（自动绑定到该企业）

**注意事项：**
- 一般 B2B 单是月结，客服在这边录单，财务月底统一对账
- 跟散客订单的字段差不多，多了个「企业」字段

### 3.7 收款记录

**入口**：左边「收款记录」

**这个页面是干什么的**：所有**已完成订单**的收款汇总，相当于现金流流水账。

**页面显示：**
- 顶部「**实收合计**」：选定时间段内的总收款
- 每行：订单号 + 客户 + 报价 / 实收 / 定金 / 付款方式

**功能：**
- 按月筛选
- 按付款方式（现金 / 银行转账）筛选
- 找到「**未付款**」状态的订单 → 客户欠款追款

**注意事项：**
- 这里的数字是**师傅在师傅端填账单后**自动同步过来的，客服一般不修改
- 如果发现金额不对（比如师傅填错了），让管理员去订单详情手动修

---

### 3.8 提货安装（IKEA）

**入口**：左边「提货安装」

**这个页面是干什么的**：IKEA 提货、配送、安装这类单独的业务流。

**和「全部订单」的区别：**
- 「全部订单」只显示**搬家**类订单
- 「提货安装」只显示**IKEA**类订单（带 IKEA 订单号、门店、商品清单等专门字段）

**录入入口**：「新建订单」→ 服务类型选 「IKEA 提货」

**特有字段：**
- IKEA 订单号
- 提货门店（Rhodes / Tempe / Marsden Park / 自填）
- IKEA 二维码截图（让师傅到店核对）
- 商品清单（用户填写）
- 是否需要垃圾清运

---

### 3.9 清洁业务

**入口**：左边「清洁业务」

**这个页面是干什么的**：所有清洁服务订单（保洁、退租清洁、深度清洁等）。

**状态比搬家少：**
- 待报价 → 客户填的需求，客服看完后报价
- 待确认 → 报价后等客户确认
- 进行中
- 已完成

**特有字段：**
- 清洁类型（日常/退租/深度）
- 房型（1 房/2 房/House等）
- 清洁地址（一个地址，不像搬家是两个）

---

### 3.10 寄存业务

**入口**：左边「寄存业务」

**这个页面是干什么的**：客户暂存物品的业务（按周收费）。

**状态分类：**
- **寄存中** → 物品在仓库
- **即将到期** → 7 天内到约定取件日，提醒客户取件
- **已逾期** → 超过约定日还没取，需要追加费用
- **已取件** → 完结

**特有字段：**
- 入库日 / 出库日
- 寄存周数
- 纸箱数 / 家具件数
- 每周费用

**寄存费率（已写在系统里）：**
- weeks ≤ 5：纸箱 $5/箱/周、家具 $10/件/周
- weeks > 5：纸箱 $3/箱/周、家具 $7/件/周
- 运输费另计

---

### 3.11 二手物品

**入口**：左边「二手物品」

**这个页面是干什么的**：客户提交不要的二手家具，公司评估后回收 / 转卖。

**两个标签页：**
- **客户提交**：客户从前台提交的物品（待评估）
  - 状态：待评估 → 处理中 → 已售出
- **在售商品**：公司收回来后挂牌出售的（暂未上线）

**特有操作：**
- 评估后给客户报价（客户接受才回收）
- 上架二手平台（暂时未启用）

## 5. 常见问题 FAQ

### ❓ 客户说没收到短信验证码怎么办？
- 让客户检查手机号是否输对
- 让客户重发一次（前端有「重新发送」按钮，60 秒冷却）
- 还不行 → 客服直接在后台帮他建单（绕过验证码流程）

### ❓ 师傅说他没看到派给他的单？
- 让师傅**强制刷新**师傅端页面
- 后台「派单管理」里确认这单是不是派给他了（可能派错给别人）
- 检查师傅账号是不是开启状态（系统设置 → 员工账号）

### ❓ 客户要改时间，但师傅已经派好了？
- 先改时段（订单详情 → 编辑 → 时段）
- 改完通知师傅，让他刷新
- 如果是当天临时改，建议直接电话通知师傅

### ❓ 客户问能不能加楼梯费 / 重物费 / 高速费？
- **报价是起步价**，客服在订单里可以手动改最终报价
- 师傅那边在填账单时会按实际情况加（楼梯/重物/高速等系统自动算）
- 客户看到的报价是「**起步价**」，最终以师傅现场结算为准 —— 这点接客户电话时主动说清楚，避免争议

**重物费核实流程（重要！）：**
1. 客户**第一次咨询**就主动问有没有重物（钢琴 / 双开门冰箱 / 大理石 / 保险箱 / 商用大件等，参考**第 4 节价目表**）
2. 客户**已下单**的话，必须再做一次**二次询问** —— 让客户用微信发图片或视频确认是什么东西
3. 客户报得出 + 客服能从图片/微信确认 → 按价目表加重物费
4. **客户讲不清 / 没法发图** → 在订单备注写「现场由师傅按实际情况判断」，让师傅到现场按重量+体积定价

### ❓ 客户要退定金？
**退款规则：**
- ✅ **可以退**：搬家开始时间**前 3 天以前**取消的，全额退定金
- ❌ **不能退**：搬家开始时间**前 3 天内** + **搬家当天**取消的，定金不退
  - 原因：这个时间段我们已经排好师傅 + 车辆，临时取消会影响其他客户的安排

**操作流程**：
- 跟管理员（Phill）确认是否符合退款条件
- 同意退 → 在「收款记录」里手动标记退款（备注退款原因 + 操作人）
- 把订单状态改为「已取消」，原因填「客户取消，已退定金」
- 不退 → 客户取消时取消原因填「不可退区间客户取消」，定金不动
